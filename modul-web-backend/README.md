# Modul 3 – Backend & REST API dengan Node.js 🚀

Selamat datang di **Modul Backend**! Di modul ini kamu akan naik level dari pengembangan **frontend modern** (React, Next.js, Tailwind) ke dunia **backend**: memahami bagaimana server bekerja, apa itu REST API, dan bagaimana JavaScript bisa berjalan di luar browser menggunakan **Node.js**.

Modul ini dirancang sebagai lanjutan dari:

- Modul 1 – Web Foundations (HTML, CSS, JavaScript dasar)
- Modul 2 – Web Modern (Library vs Framework, React, Next.js, Tailwind CSS)

Di Modul 3, fokus kita adalah memahami **fondasi backend** sebelum nanti lanjut ke database, autentikasi, dan integrasi full‑stack di modul berikutnya.

## Apa yang akan dipelajari?

1. **Konsep Backend & Client–Server**  
   Memahami perbedaan frontend vs backend, bagaimana browser berkomunikasi dengan server, dan di mana posisi Node.js dalam arsitektur web modern.

2. **Node.js & Ekosistem npm**  
   Mengenal Node.js sebagai runtime JavaScript di server, cara inisialisasi project backend, dan bagaimana menggunakan package dari npm untuk membangun aplikasi.

3. **REST API Dasar dengan Express**  
   Membangun REST API sederhana (tanpa database) dengan Node.js dan Express: membuat endpoint, mengelola request & response, dan mengimplementasikan operasi CRUD menggunakan data in‑memory.

## Hubungan dengan Modul Sebelumnya

- Di **Modul 1**, kamu sudah belajar bagaimana membangun halaman web statis dan interaktif dengan HTML, CSS, dan JavaScript.
- Di **Modul 2**, kamu mengenal cara membangun UI modern dengan React, Next.js, dan Tailwind CSS.

Sekarang, di **Modul 3**, kamu akan melihat bagaimana:

- Browser (frontend) mengirim **HTTP request** ke backend.
- Backend (Node.js) merespons dengan **JSON API** yang bisa dikonsumsi oleh aplikasi React/Next.
- Konsep di roadmap seperti **Node.js → CLI → APIs** mulai dipraktikkan secara nyata.

Selamat belajar, dan selamat datang di dunia **backend dengan Node.js**! 🚀
