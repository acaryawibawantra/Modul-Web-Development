## Apa yang akan dipelajari?
1. **HTML**: Membangun kerangka dan struktur halaman web.
2. **CSS**: Memberikan gaya, warna, dan tata letak yang menarik.
3. **JS**: Menambahkan logika dan interaktivitas pada website.
