# Final Project – Bangun & Deploy Web Portofolio Pribadi

Selamat datang di **Final Project** Web Development Workshop! Ini adalah puncak dari semua yang sudah kamu pelajari — dari HTML pertama sampai deploy ke internet.

**Tugasmu:** membangun **website portofolio pribadi** dari nol, lalu men-deploy-nya sampai **live** dan bisa diakses siapa saja.

Portofolio ini bukan sekadar tugas — ini karya nyata yang bisa kamu pajang di **CV, LinkedIn, atau lamaran kerja/magang**. 🚀

---

## 1. Apa yang Harus Dibuat

Sebuah **website portofolio pribadi** yang berisi minimal:

- **Nama & foto/avatar** — siapa kamu.
- **Tentang diri (About)** — deskripsi singkat tentang kamu.
- **Skill** — teknologi/kemampuan yang kamu punya.
- **Minimal 1 project/karya** — bisa project dari workshop ini, tugas kuliah, atau apa pun.
- **Kontak** — email, LinkedIn, GitHub, atau media sosial lain.

Isi boleh kamu kembangkan sesukamu — tambahkan section pengalaman, sertifikat, blog, dll.

---

## 2. Bebas Pilih Teknologi

Kamu **bebas** memakai teknologi apa pun yang sudah dipelajari di workshop. Pilih sesuai level kenyamananmu:

| Level | Teknologi | Cocok untuk | Modul rujukan |
| --- | --- | --- | --- |
| **Dasar** | HTML + CSS + JavaScript | Yang ingin simpel & solid | Modul 1.1 HTML, 1.2 CSS, 1.3 JS |
| **Menengah** | React + Tailwind CSS | Yang ingin komponen & styling modern | Modul 2.2 React, 2.4 Tailwind |
| **Lanjutan** | Next.js + Tailwind CSS | Yang ingin tingkat profesional | Modul 2.3 Next.js, 2.4 Tailwind |

Tidak ada nilai lebih hanya karena pakai teknologi lebih rumit. **Portofolio HTML/CSS yang rapi lebih baik daripada Next.js yang berantakan.** Pilih yang bisa kamu selesaikan dengan baik.

---

## 3. Mulai dari Nol — Langkah demi Langkah

### Kalau pakai HTML/CSS/JS (Dasar)

Tidak perlu install apa pun. Cukup:

1. Buat folder project baru, misal `portofolio`.
2. Buat file `index.html`, `style.css`, dan `script.js`.
3. Mulai ngoding. Buka `index.html` langsung di browser untuk melihat hasilnya.

> Rujukan: Modul 1.1 HTML, 1.2 CSS, 1.3 JS.

### Kalau pakai React + Tailwind (Menengah)

1. Pastikan **Node.js** sudah terinstall (cek dengan `node -v`).
2. Buat project React dengan Vite:
```
npm create vite@latest portofolio – –template react

cd portofolio

npm install
```

3. Install & setup Tailwind CSS (ikuti Modul 2.4).
4. Jalankan `npm run dev` dan mulai ngoding.

> Rujukan: Modul 2.2 React, 2.4 Tailwind.

### Kalau pakai Next.js + Tailwind (Lanjutan)

1. Pastikan **Node.js** sudah terinstall.
2. Buat project Next.js (pilih "Yes" saat ditanya Tailwind):

```
npx create-next-app@latest portofolio

cd portofolio

npm run dev
```

3. Mulai ngoding di folder `app/`.

> Rujukan: Modul 2.3 Next.js, 2.4 Tailwind.

## 4. Deploy Portofolio Kamu

Setelah portofolio selesai, **deploy sampai live** mengikuti materi modul 5:

1. **Persiapan & push ke GitHub** → Modul 5.1
2. **Deploy ke Vercel** → Modul 5.2

Setelah deploy, kamu akan punya URL publik seperti `https://portofolio-namamu.vercel.app`. **Itulah yang kamu submit.**

> Kalau portofoliomu punya backend/database, lihat Modul 5.3 (opsional).

---

## 7. Tips

- **Mulai sederhana, perbaiki bertahap.** Bikin versi paling dasar dulu sampai live, baru dipercantik. Lebih baik portofolio sederhana yang live daripada yang mewah tapi tidak selesai.
- **Deploy lebih awal.** Jangan tunggu sempurna. Deploy versi awal, lalu tinggal `git push` tiap ada update (auto-deploy).
- **Cek di HP.** Banyak orang buka portofolio dari HP — pastikan tampilannya tidak berantakan.
- **Isi dengan yang jujur.** Portofolio ini akan kamu pakai beneran, jadi isi dengan skill & project yang benar-benar kamu punya.
